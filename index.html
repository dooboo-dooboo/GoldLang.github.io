문법

Script->를 첫 줄에, <-Script를 마지막 줄에 쓴다.

한 문장의 코드는 한 줄에 쓰며, 들여쓰기나 뒤에 추가 띄어쓰기는 허용되지 않는다.

Show <content> 는 content를 출력하는 코드이다.

content에는 문자열, 숫자 등을 입력할 수 있다.

실수 변수 선언과 변경은 N a <- 10 와 같이 쓴다.

문자열 변수 선언과 변경은 S a <- hello 와 같이 쓴다.

Bool형 변수 선언과 변경은 B a <- 0 와 같이 쓰며, 0은 거짓, 1은 참으로 간주한다.

Go 23 와 같이 쓰면 원하는 줄인 23줄로 이동한다.

If a Do 23 와 같이 쓰면 a가 1일 때 23줄로 이동하고, a가 0이면 바로 다음 코드를 진행한다.
